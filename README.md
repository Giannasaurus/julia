# how to run
julia in cmd:
```
C:\current\dir> julia
```
julia repl:
```
julia>
```
locate file:
> [!TIP]
> right-click anywhere to paste text
> to enable: title bar, right-click > properties > QuickEdit Mode
```
julia> cd("path\to\julia\file")
```
run file:
```
julia> include("julia-file.jl");
```
